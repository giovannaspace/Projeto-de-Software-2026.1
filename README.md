# My Bookshelf

Repositório para a disciplina de Projeto de Software  
Engenharia de computação - UFAL

### Funcionalidades do My Bookshelf - Minha Estante
-> Operações de Cadastro e Login;
-> Catálogo de livros para navegação e seleção;
-> Diferenciar tipos de leitura: Audiobook e Livro Físico / Ebook (avaliar narração/efinir empréstimo)
-> Incentivo via e-mail caso o usuário realize o login após um período maior que 30 dias;
-> Biblioteca pessoal do usuário: adição de livros no perfil;
-> Adicionar avaliações para livros da biblioteca (nota de 0 a 5 ou resenha literária);
-> Criação de categorias para os livros salvos na biblioteca;
-> Adicionar etiquetas específicas para livros salvos na biblioteca;
-> Definição de metas literárias mensais ou anuais;
-> Armazenar datas de início e conclusão de leitura para livros salvos;
-> Ver retrospectiva (livros lidos em determinado ano);
-> Acompanhar progresso de leitura;


'git clone <url do repositório>'  
'python -m venv venv'  
'source venv/bin/activate'  
'pip install -r requirements.txt'  
'python main.py'
